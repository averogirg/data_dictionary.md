# Описание данных для проекта Superstore Analysis

## Источник данных
Данные взяты с Kaggle: "Superstore Sales Dataset"
Период: 2016-2017 гг.
Объем: 200 строк (упрощенная версия для демонстрации)

## Структура таблицы

### 1. Идентификационные поля
- **Row ID** - номер строки в исходном файле
- **Order ID** - уникальный ID заказа (формат: "CA-2017-152156")
- **Customer ID** - уникальный ID клиента (формат: "CG-12520")

### 2. Информация о заказе  
- **Order Date** - дата оформления заказа (ДД/ММ/ГГГГ)
- **Ship Date** - дата доставки
- **Ship Mode** - способ доставки:
  * Standard Class
  * Second Class  
  * First Class
  * Same Day

### 3. Информация о клиенте
- **Customer Name** - имя клиента
- **Segment** - сегмент клиента:
  * Consumer (потребитель)
  * Corporate (корпоративный)
  * Home Office (домашний офис)

### 4. Географические данные
- **Country** - страна (только United States)
- **City** - город
- **State** - штат (например, California, Kentucky)
- **Postal Code** - почтовый индекс
- **Region** - регион (South, West, East, Central)

### 5. Информация о товаре
- **Product ID** - уникальный ID товара
- **Category** - категория товара:
  * Furniture (мебель)
  * Office Supplies (канцелярия)  
  * Technology (техника)
- **Sub-Category** - подкатегория товара
- **Product Name** - название товара

### 6. Финансовые показатели
- **Sales** - сумма продажи в долларах
  * Тип: десятичное число
  * Пример: 261.96
  * Использование: основная метрика для анализа

## Правила работы с данными
1. Все суммы в долларах США (USD)
2. Даты в формате ДД/ММ/ГГГГ
3. Страна только "United States"
4. Отсутствующие значения: нет
