# Данные

**`superstore_sample.csv`** — основная таблица с продажами для анализа. Содержит информацию о заказах, клиентах и товарах.

**`supplier.txt`** - таблица с поставщиками. Содержит информацию о рейтинге, категории поставщика и его регион.

Источник: [Superstore Sales Dataset на Kaggle](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting)
