# Данные по таблицам

В этой папке хранятся исходные таблицы, необходимые для проведения анализа.

## Описание файлов

*   **`superstore_sample.csv`** — основная таблица с продажами для анализа. Содержит информацию о заказах, клиентах, товарах, доставке и географии продаж. Это выгрузка из датасета Superstore с Kaggle.
*   **`suppliers.csv`** — дополнительная таблица с поставщиками. Содержит информацию о названии компании, рейтинге, категории товаров и регионе (стране) поставщика. Создана вручную для демонстрации операций объединения таблиц (JOIN).

## Связь между таблицами

Таблицы могут быть объединены для анализа по общему полю **`category`** (категория товаров).
*   В таблице `superstore_sample.csv` поле `category` указывает категорию проданного товара.
*   В таблице `suppliers.csv` поле `category` указывает категорию товаров, с которой работает поставщик.

---

**Источник основного датасета:** [Superstore Sales Dataset на Kaggle](https://www.kaggle.com/datasets)
